## 本次更新（四大改造）
### 1. 首页（合并工作台+数据看板）
`dashboard.html` 现为「首页」，三个标签页：
- 个人概览：统计卡 / 快捷操作 / 最近案例 / 推荐产品
- 团队数据：原数据看板全部图表（趋势/风格·空间分布/复用 Top10/部门统计）
- 复用流水：团队案例复用动态 + 顶部通知中心
左侧导航已删除「工作台」「数据看板」，合并为「首页」。`analytics.html` 已移除。

### 2. 产品库（总库 + 个人库 + 对比）
- 每个产品卡片有「添加」「对比」。
- 我的产品库：可自定义文件夹（如「非洲风格」），把商品添加到文件夹；每个文件夹「一键下载全部」。
- 对比：勾选 2 个产品，弹窗按属性逐行对比并高亮差异。
- 数据持久化在 localStorage（`gd-prod-folders`）。

### 3. 案例卡片统一（案例库 / 我的案例 / NAS）
统一为 `gdcase.js` 一套卡片 + 详情：详情文件（产品/模型/CAD）、VR 或案例图、案例对比、一键复用、导出选品清单(BOM)。
差异：案例库=浏览+复用；我的案例=本人案例并多「编辑/删除」（就地更新、删除二次确认）；NAS=按目录浏览并多「下载到我的NAS」。

### 4. 系统管理并入个人中心 + 新增模块
- 个人中心 `profile.html` 标签：我的成就（等级/积分/徽章/明细）、数字分身、同步设置、系统管理（用户管理/权限矩阵/系统设置/审计日志）。原 `admin.html`、`settings.html` 已移除。
- 设计师榜单 `ranking.html`：复用榜/产量榜/积分榜 × 本月/本季度/本年，前三名领奖台 + 完整排行。
- 团队协作 `team.html`：项目卡（进度/成员/客户）+ 任务看板（待开始/进行中/待审核/已完成）。
- 通知中心：首页🔔，未读数显示在侧栏「首页」徽标。

## 共享模块
- `common.js`：侧栏渲染（唯一来源）、暗色/历史/快捷键/收藏/对比/下载、真实图片、持久化（GDStore）、个人产品库（GDLib）、产品对比、复用副本、导航。
- `gdcase.js`：统一案例卡片 + 详情/VR/图片/PPT/对比/复用/BOM/编辑删除。

## 部署
推到 GitHub 仓库根目录（含 `.nojekyll`、`data/`），Settings→Pages 选 main 分支根目录。入口 `index.html`（资源库门户）；后台首页 `dashboard.html`（登录后由门户「控制台」进入）。
